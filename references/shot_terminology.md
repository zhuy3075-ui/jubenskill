## 景别 Shot Size

### 基础景别

| 缩写 | 英文全称 | 中文 | 描述 | 用途 |
|------|----------|------|------|------|
| **ECU** | Extreme Close-Up | 大特写 | 仅显示面部局部(眼睛/嘴唇) | 强调情绪、细节 |
| **CU** | Close-Up | 特写 | 头部和肩膀 | 表情、反应 |
| **MCU** | Medium Close-Up | 中近景 | 胸部以上 | 对话、采访 |
| **MS** | Medium Shot | 中景 | 腰部以上 | 常规对话 |
| **MLS** | Medium Long Shot | 中远景 | 膝盖以上 | 动作+环境 |
| **LS** | Long Shot | 远景/全身 | 全身+部分环境 | 展示动作 |
| **ELS** | Extreme Long Shot | 大远景 | 小人物+大环境 | 建立场景 |
| **VLS** | Very Long Shot | 超远景 | 人物极小 | 史诗感 |

### 特殊景别

| 缩写 | 英文全称 | 中文 | 描述 |
|------|----------|------|------|
| **OTS** | Over-the-Shoulder | 过肩镜头 | 从一人肩膀后拍另一人 |
| **POV** | Point of View | 主观镜头 | 角色视角 |
| **2S** | Two Shot | 双人镜头 | 两人同框 |
| **3S** | Three Shot | 三人镜头 | 三人同框 |
| **GS** | Group Shot | 群戏镜头 | 多人同框 |
| **OSS** | Over-the-Shoulder Single | 单人过肩 | 焦点在对话者 |
| **INS** | Insert | 插入镜头 | 细节特写(手、物品) |
| **EST** | Establishing Shot | 建立镜头 | 场景开场全景 |

### 景别提示词格式

```
[shot_size] of [subject], [additional framing notes]
```

示例：
```
close-up of woman's face, slight angle, soft focus background
medium shot of man standing, full body visible, urban background
extreme close-up of eyes, reflection visible, intense gaze
over-the-shoulder shot, focus on speaker, blurred listener
```

## 运镜 Camera Movement

### 基础运镜

| 术语 | 英文 | 描述 | 效果 |
|------|------|------|------|
| **固定** | Static | 摄影机不动 | 稳定、客观 |
| **推** | Push In / Dolly In | 向主体靠近 | 强调、紧张 |
| **拉** | Pull Out / Dolly Out | 远离主体 | 揭示、孤独 |
| **摇** | Pan | 水平旋转 | 跟随、环视 |
| **俯仰** | Tilt | 垂直旋转 | 上下打量 |
| **移** | Dolly / Track | 平行移动 | 跟随动作 |
| **跟** | Tracking / Following | 跟随主体 | 动态感 |
| **升降** | Crane / Jib | 垂直升降 | 戏剧性 |
| **环绕** | Orbit / Arc | 围绕主体旋转 | 360度展示 |

### 进阶运镜

| 术语 | 英文 | 描述 | 效果 |
|------|------|------|------|
| **手持** | Handheld | 手持拍摄 | 紧张、真实 |
| **稳定器** | Steadicam | 稳定器拍摄 | 流畅跟随 |
| **变焦** | Zoom | 光学变焦 | 快速强调 |
| **甩镜** | Whip Pan | 快速摇摄 | 时间跳跃 |
| **滑轨** | Slider | 短距离平移 | 微妙动态 |
| **航拍** | Aerial / Drone | 无人机拍摄 | 宏大视角 |
| **滚转** | Dutch Angle Roll | 倾斜滚动 | 混乱、不安 |

### 复合运镜

| 术语 | 英文 | 描述 |
|------|------|------|
| 推摇 | Push + Pan | 靠近同时水平旋转 |
| 跟升 | Track + Crane | 跟随同时上升 |
| 环推 | Orbit + Push | 环绕同时靠近 |

### 运镜提示词格式

```
[movement_type] shot, [direction/path], [speed], [subject tracking]
```

示例：
```
slow push in towards character's face, building tension
smooth tracking shot following character walking, side angle
crane shot rising up to reveal cityscape, dramatic
handheld camera following action, documentary style
```

## 镜头角度 Camera Angle

### 垂直角度

| 角度 | 英文 | 描述 | 心理效果 |
|------|------|------|----------|
| **鸟瞰** | Bird's Eye / Top-Down | 正上方俯拍 | 全知视角 |
| **俯拍** | High Angle | 高于眼平线向下 | 渺小、脆弱 |
| **平视** | Eye Level | 与眼睛平齐 | 中立、平等 |
| **仰拍** | Low Angle | 低于眼平线向上 | 强大、威胁 |
| **蚁视** | Worm's Eye | 地面向上 | 极度威严 |

### 水平角度

| 角度 | 英文 | 描述 |
|------|------|------|
| **正面** | Front / Head-On | 直视镜头 |
| **3/4侧** | Three-Quarter | 约45度侧面 |
| **侧面** | Profile | 90度侧面 |
| **背面** | Back / Rear | 从背后拍摄 |

### 特殊角度

| 角度 | 英文 | 描述 |
|------|------|------|
| **荷兰角** | Dutch Angle / Canted | 倾斜画面 |
| **倒置** | Inverted | 上下颠倒 |

## 转场 Transitions

### 基础转场

| 转场 | 英文 | 描述 | 时间感 |
|------|------|------|--------|
| **硬切** | Cut | 直接切换 | 即时 |
| **淡入** | Fade In | 从黑/白渐显 | 开始 |
| **淡出** | Fade Out | 渐变到黑/白 | 结束 |
| **叠化** | Dissolve / Cross Dissolve | 两画面渐变叠加 | 时间流逝 |
| **划变** | Wipe | 一画面推开另一画面 | 场景转换 |

### 进阶转场

| 转场 | 英文 | 描述 |
|------|------|------|
| **匹配剪辑** | Match Cut | 相似形状/动作衔接 |
| **跳切** | Jump Cut | 同场景时间跳跃 |
| **碰撞剪辑** | Smash Cut | 强烈对比直切 |
| **L型剪辑** | L-Cut | 画面先切声音延续 |
| **J型剪辑** | J-Cut | 声音先入画面后切 |
| **闪白** | Flash / White Out | 闪烁到白色 |
| **闪黑** | Dip to Black | 快速黑场 |

## 构图术语 Composition

### 构图规则

| 术语 | 英文 | 描述 |
|------|------|------|
| **三分法** | Rule of Thirds | 画面分9格，主体在交点 |
| **对称** | Symmetry | 左右/上下对称 |
| **引导线** | Leading Lines | 线条引导视线 |
| **框中框** | Frame within Frame | 用前景元素框住主体 |
| **负空间** | Negative Space | 大量留白 |
| **深度** | Depth | 前中后景层次 |

### 人物位置

| 术语 | 英文 | 描述 |
|------|------|------|
| **居中** | Centered | 主体在画面中央 |
| **左三分** | Left Third | 主体在左侧1/3处 |
| **右三分** | Right Third | 主体在右侧1/3处 |
| **头部空间** | Headroom | 头顶到画面顶的空间 |
| **视线空间** | Looking Room | 视线方向的留白 |

| 镜头 | 英文 | 描述 |
|------|------|------|
| **反应镜头** | Reaction Shot | 角色对事件的反应 |
| **切入镜头** | Cutaway | 切到相关但不同的画面 |
| **空镜头** | Empty Frame | 无人物的环境镜头 |
| **主观镜头** | Subjective Shot | 角色第一人称视角 |
| **客观镜头** | Objective Shot | 旁观者视角 |
| **定场镜头** | Master Shot | 场景全景作为参考 |
| **慢动作** | Slow Motion | 减速播放 |
| **快动作** | Fast Motion | 加速播放 |
| **定格** | Freeze Frame | 画面静止 |
| **分屏** | Split Screen | 画面分割显示多个内容 |
