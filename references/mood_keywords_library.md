## 基础情绪词库

### 正面情绪

| 情绪 | 英文关键词 | 视觉表现 |
|------|-----------|----------|
| **快乐** | happy, joyful, cheerful, delighted | bright lighting, warm colors, smiles |
| **兴奋** | excited, thrilled, energetic, dynamic | vibrant colors, dynamic angles |
| **平静** | calm, peaceful, serene, tranquil | soft lighting, muted tones, still |
| **温暖** | warm, cozy, comfortable, tender | golden light, earth tones |
| **希望** | hopeful, optimistic, inspiring | sunrise light, upward angles |
| **浪漫** | romantic, intimate, loving, tender | soft focus, warm pink/red tones |
| **自信** | confident, powerful, strong | low angle, strong lighting |
| **满足** | content, satisfied, fulfilled | soft smile, relaxed posture |

### 负面情绪

| 情绪 | 英文关键词 | 视觉表现 |
|------|-----------|----------|
| **悲伤** | sad, melancholic, sorrowful, grief | blue tones, rain, downcast |
| **愤怒** | angry, furious, enraged, hostile | red tones, sharp shadows, close-up |
| **恐惧** | fear, scared, terrified, anxious | dark shadows, low light, distortion |
| **孤独** | lonely, isolated, abandoned | empty space, single figure, cold tones |
| **绝望** | desperate, hopeless, despair | harsh shadows, dark, trapped |
| **焦虑** | anxious, nervous, worried, tense | handheld, tight framing, shadows |
| **嫉妒** | jealous, envious | green tint, observing angle |
| **内疚** | guilty, ashamed, remorseful | downward gaze, shadows on face |

### 中性/复杂情绪

| 情绪 | 英文关键词 | 视觉表现 |
|------|-----------|----------|
| **神秘** | mysterious, enigmatic, cryptic | fog, shadows, partial reveal |
| **怀旧** | nostalgic, wistful, reminiscent | warm sepia, soft focus, vintage |
| **沉思** | contemplative, pensive, thoughtful | profile shot, window light |
| **紧张** | tense, suspenseful, uneasy | tight framing, harsh shadows |
| **困惑** | confused, perplexed, puzzled | tilted angle, mixed lighting |
| **期待** | anticipation, expectation | forward lean, eyes focused |
| **疏离** | distant, detached, alienated | wide shot, cold tones, barriers |

## 提示词组合模板

### 基础氛围模板

```
[emotion] mood, [lighting_type] lighting, [color_tone] tones,
[atmosphere_descriptor], [visual_style]
```

### 示例组合

**浪漫场景**：
```
romantic intimate mood, golden hour soft lighting, warm pink and orange tones,
dreamy atmosphere with soft focus, cinematic beautiful
```

**紧张场景**：
```
tense suspenseful mood, harsh dramatic lighting, cold blue-gray tones,
claustrophobic atmosphere with sharp shadows, noir style
```

**悲伤场景**：
```
melancholic sorrowful mood, overcast diffused lighting, muted blue tones,
heavy somber atmosphere, quiet and still, cinematic emotional
```

**恐怖场景**：
```
eerie terrifying mood, low key harsh lighting, dark desaturated tones,
oppressive unsettling atmosphere with deep shadows, horror style
```

**欢乐场景**：
```
joyful cheerful mood, bright natural lighting, warm vibrant colors,
lively energetic atmosphere, dynamic and colorful
```
