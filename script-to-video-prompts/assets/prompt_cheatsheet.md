## ⚡ 快速模板

### 建立镜头
```
extreme long shot, establishing shot of [场所],
[时间] lighting, [天气], cinematic, high quality
```

### 人物特写
```
close-up of [人物描述], [表情],
[光线], bokeh background, cinematic portrait
```

### 对话双人
```
medium two-shot of [人物A] and [人物B],
[互动描述], [场所], [光线], natural, cinematic
```

### 跟随镜头
```
tracking shot following [人物],
[动作], [场所], smooth motion, cinematic
```
